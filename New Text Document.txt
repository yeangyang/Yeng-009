ບົດທີ 1 ການໃນໍາໃຊ້ Bootstrap 5

1.Bootstrap ແມ່ນຫຍັງ?.

Bootstrap ແມ່ນ Frameword ທີ່ຖືກອອກບບມາເພື່ອໃຫ້ງ່າຍໃນການພັດທະນາເວັບໄຊ ແລະ ເຮັດໃຫ້ເວັບໄຊມີຄວາມສວຍງາມທັນສະໄໝ່ຍິ່ງຂື້ນດວ້ຍການນໍາໃຊ້ Class ຕ່າງໆທີ່ມີ ໃຫ້ເລືອກຢ່າງຫຼວງຫຼາຍ,ເໝາະສໍາລັບຜູ້ທີ່ເລີ່ມຕົ້ນໃນການພັດທະນາເວັບໄຊ.

2. ການຕິດຕັ້ງ Bootstrap

ຜູ້ນໍາໃຊ້ສາມາດເຂົ້າໄປດາວໂຫຼດເອົາຟຣາຍຂອງ Bootstrap ໄດ້ທີ ເວັບໄຊ getbootstrap.com ຫຼື ສາມາດນໍາໃຊ້ BS ແບບອອນໄລກໍໄດ້ເຊັ່ນດຽວກັນ.

3. ຮູບແບບຂອງການເອີ້ນໃຊ້ Bootstrap (ກໍລະນີທີ່ເຮົາດາວໂຫຼດຟາຍມານໍາໃຊ້)

<meta charset="UTF-8">  ເຮັດໃຫ້ເວບບາວເຊີຮອງຮັບພາສາ
<meta name="viewport" content="width=device-width, initial-scale=1">  ເຮັດໃຫ້ການສະແດງ
<link rel="stylesheet" href="bootstrap5/css/bootstrap.min.css">  ເອີ້ນເອົາ CSS ຂອງ
<script src="bootstrap5/js/bootstrap.bundle.min.js"></script> ເອີ້ນເອົາ js ຂອງ
